# Tuberculosis-Burden-by-Country-Global-Analytics-Dashboard
Tuberculosis Burden by Country : Global Analytics Dashboard

Designed and built a multi-view Tableau dashboard analyzing the global Tuberculosis (TB) burden across 219 countries over a 24-year span using the WHO TB dataset.

• Integrated 12 analytical visualizations covering incidence, prevalence, mortality, TB-HIV co-infection rates, case detection gaps, and regional trends : enabling country-level and longitudinal comparisons in a single interactive interface.
• Performed end-to-end data preparation including cleaning, feature renaming, missing value handling, and restructuring country-year records to support time-series analysis at scale.
• Surfaced under-reporting patterns by visualizing case detection gaps, helping identify regions with critical public-health blind spots.

Tech: Tableau · WHO TB Dataset · Data Cleaning · Calculated Fields · Public Health Analytics
